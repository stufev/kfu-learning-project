# Курсы повышения квалификации

## Стек
- Vue 3
- Pinia
- Vue Router
- Node.js
- Express
- Prisma
- SQLite

## Функционал
- регистрация и авторизация (JWT)
- просмотр курсов
- запись на курсы
- личный кабинет
- статусы прохождения
- обратная связь

## Запуск

backend:
npm install
npx prisma migrate dev
node src/index.js

frontend:
npm install
npm run dev
